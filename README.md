# Gerador de senhas em python

um projeto simples que fiz no tédio, ele gera senhas aleatorias de 12 caracteres que incluem letras minusculas e maiusculas e também
incluem numeros

## tecnologias

● Python

## exemplo de como fica no terminal

'''bash

[Gerador de senhas automaticas]
gerando senha...
senha: gvZKdy94Gzfg

'''
